Introduction

Are you tired of taking handwritten notes for online classes? What if you could print them easily without the hassle of editing manually?

This Python script automates the process of merging class slides, formatting them beautifully, and optimizing their layout for A4-size pages. It also allows color inversion for better printing results.

Why Choose Merger?

Despite being Free and Open-source, Merger has-

High Performance: Merger is designed with cutting-edge optimization techniques that works intelligently into layers, making it the fastest solution for merging and formatting slides for printout.

Print-Ready Output: Specially tailored for class slides (16:9), it ensures an optimal layout on A4 pages, maximizing clarity and readability.

Feature-Packed: With built-in color inversion, slide bordering, page numbering, strategy choice, 1st slide removal with indicator, custom cover page and work summary - it covers all essential needs without additional tools.

How to Use

Using this script is simple:

  • Place your PDF files into the Input folder.
  • Run the script and choose your preferences.
  • The processed files will be saved in the Output folder.

Available Options:

  • Merge PDFs: Combine all PDF files in the input folder into a single PDF.
  • Merge class slides: The core feature of this script which makes class slides printable. It includes:
    • Slide count in a page: Select how many slides should be appeared on each page.
    • Remove 1st slides: Choose if you want to remove the typical cover slide of each pdf.
      • Indicate beginning of new pdf: You can add star symbols in the margins to indicate initialization of new pdf's or lectures. Best alternative to cover slides- saving page space and printing cost.
    • Select strategy: Vector Assembly for best print quality, Pixel Construct for minimal loading time in digital view. To print Vector Assembly mode's output, consider opening the pdf using sumatra and then you can print using adobe pdf print.
    • Select Output DPI: Choose the quality of the output in pixel construct mode.
    • Color inversion: Toggle this option to "Yes" if the slides have black backgrounds.
    • Cover page: Custom cover page with minimal design. Includes input fields for Title, Chapter No, Subject and Student Name.

Setup Instructions

Windows

  1. Install Python:

    Download Python from the official website. During installation, ensure you select the "Add Python to PATH" option, then proceed with the default settings.

  2. Download & Set Up the Libraries:

    Download the script from here. Extract the archive to a folder.

    Open cmd and run the following command:

     pip install mergerz
  3. Run the Script:

    Launch Merge.py.

Linux

  1. Download & Set Up the Libraries:

    Download the archive from here. Extract the archive to a folder.

    Open terminal in the folder and run following commands:

     python3 -m venv myenv 
    
            
     source myenv/bin/activate 
    
		    
     pip install mergerz
  2. Run the Script:

    Make sure virtual environment is activated and run following command in the folder:

     python3 Merge.py

Android

Though we will be using a Linux distro (Ubuntu) in Android, we can't use the pre-compiled Merge.bin because Android has restrictions regarding the permissions.

  1. Install Required Apps:

    Install UserLAnd and ZArchiver from the Play Store. Open UserLAnd and install the "Ubuntu Minimal Terminal" package. Once set up, you’ll be redirected to the Ubuntu terminal interface.

  2. Set Up Python:

    Run the following command in the terminal:

     sudo apt update && sudo apt upgrade -y && sudo apt install python3 python3-pip -y && sudo pip install mergerz
  3. Download & Place the Script:

    Download the script from here. Use ZArchiver to extract the zip to /sdcard/Android/data/tech.ula/files/ (this files folder must contain Merge.py, Input, Output, static, Temp)

    Note: You have to manually navigate to this location to place input files and retrieve output files.

  4. Run the Script:

    Open the UserLAnd terminal and run:

    cd /sdcard/Android/data/tech.ula/files/ && sudo python3 Merge.py

    This will launch the script. Save this command for future use. To exit, select "Exit" from the notification panel. To rerun the script, reopen UserLAnd, select the Ubuntu session, and paste the saved command.
