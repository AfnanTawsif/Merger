Introduction

Are you tired of taking handwritten notes for online classes? What if you could print them easily without the hassle of editing manually?

This Python script automates the process of merging class slides, formatting them beautifully, and optimizing their layout for A4-size pages. It also allows color inversion for better printing results.

How to Use

Using this script is simple:

  • Place your PDF files into the Input folder.
  • Run the script (Merge.py) and choose your preferences.
  • The processed files will be saved in the Output folder.

Available Options:

  • Merge PDFs: Combine all PDF files in the input folder into a single PDF.
  • Merge Class Slides: The core feature of this script which makes class slides printable. It includes:
    • Quality Settings: Choose the quality of the output.
    • Slides Per Page: Select how many slides appear on each page (3 per page recommended for printing).
    • Color Inversion: Toggle this option to "Yes" if the slides have black backgrounds.

Setup Instructions

For Windows

  1. Install Python:

    Download Python 3.12.7 from the official website. During installation, ensure you select the "Add Python to PATH" option, then proceed with the default settings.

  2. Install Microsoft C++ Build Tools:

    Download from here. Launch the installer, go to the "Available" tab, and select any package to install.

    Under "Desktop & Mobile", choose "Desktop development with C++". On the right panel, check "Windows 11 SDK" along with the default selections, and click "Install".

  3. Download and Set Up the Script:

    Download the script from here. Extract the archive to folder name and open the folder. Launch the Setup_Library.bat file to install the required libraries.

    If you encounter errors during library installation, download precompiled .whl files for problematic libraries from PyPI, place them in the script's folder, and update the pip install command in the batch file using Notepad++ (replace those library names with the whl file names with extension). Save and rerun the batch file.

For Android

  1. Install Required Apps:

    Install UserLAnd and ZArchiver from the Play Store. Open UserLAnd and install the "Ubuntu Minimal Terminal" package. Once set up, you’ll be redirected to the Ubuntu terminal interface.

  2. Set Up Python:

    Run the following command in the terminal:

    sudo apt update && sudo apt upgrade -y && sudo apt install python3 python3-pip -y && sudo pip3 install PyPDF2 reportlab pymupdf colorama fitz frontend tools --timeout=120 && sudo pip3 uninstall pymupdf -y && sudo pip3 install pymupdf --timeout=120
  3. Download and Place the Script:

    Download the script from here. Use ZArchiver to extract the zip to /sdcard/Android/data/tech.ula/files/ (this files folder must contain Merge.py, log.txt, Input, Output, Temp, static)

    Note: You must manually navigate to this location to place input files and retrieve output files.

  4. Run the Script:

    Open the UserLAnd terminal and run:

    cd /sdcard/Android/data/tech.ula/files/ && sudo python3 Merge.py

    This will launch the script. Save this command for future use. To exit, select "Exit" from the notification panel. To rerun the script, reopen UserLAnd, select the Ubuntu session, and paste the saved command.

Troubleshooting

Issue: Fitz has no attribute 'open'

  • Fix for Windows: 
    pip uninstall pymupdf 
    

    pip install pymupdf
  • Fix for Android: 
    sudo pip3 uninstall pymupdf 
    

    sudo pip3 install pymupdf
